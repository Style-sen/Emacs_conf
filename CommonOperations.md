# 1. 光标移动
### 1.一次执行n次移动:(1)C-u n或M-n 重复执行n次后继命令 (2)执行移动命令；    
| 移动命令 | 作用 |
| -- | -- |
| C-f     | 光标前进一个字符       | 
| C-b     | 光标后退一个字符       | 
| C-p     | 上移一行               | 
| C-n     | 下移一行               | 
| M-f     | 前进一个单词           | 
| M-b     | 后退一个单词           | 
| C-a     | 行首                   | 
| C-e     | 行尾                   | 
| M-a     | 句首                   | 
| M-e     | 句尾                   | 
| M-{     | 上一个段落             | 
| M-}     | 下一个段落             | 
| C-v     | 屏幕上卷一个屏         | 
| M-v     | 屏幕下卷一个屏         | 
| C-M-v   | 另一个窗口下卷一个屏幕 | 
| C-M-V   | 另一个窗口上卷一个屏幕 | 
| M-<     | 文件头                 | 
| M->     | 文件尾                 |
